# https://roomle-dev.github.io/cdn-embedding-example/

Examples of loading the @roomle/embedding-lib through a CDN.

### JSDelivr

```html
<script type="module">
    import RoomleConfiguratorApi
        from "https://cdn.jsdelivr.net/npm/@roomle/embedding-lib@6.3.0/roomle-embedding-lib.js";
</script>
```

### unpkg.com

```html
<script crossorigin="anonymous" type="module">
    import RoomleConfiguratorApi from "https://unpkg.com/@roomle/embedding-lib@6.3.0/roomle-embedding-lib.js";
</script>
```
